# C / C++

The C ABI hub. A single `wickra_copilot.h` header + prebuilt library links from C,
C++, or any C-capable language. Drive a copilot with the JSON command protocol.

```bash
# prebuilt wickra_copilot.h + library per platform:
# github.com/wickra-lib/wickra-copilot/releases
```

```c
#include "wickra_copilot.h"

const char *spec = "{\"symbols\":[\"BTCUSDT\"],\"lookback\":20,"
                   "\"facts\":[\"price_move\",\"funding_flip\"]}";

WickraCopilot *c = wickra_copilot_new(spec);
const char *cmd = "{\"cmd\":\"build_context\",\"feeds\":{ /* symbol -> feed */ }}";
char *context = wickra_copilot_command(c, cmd);   // JSON response

printf("%s\n", context);
wickra_copilot_free(c);
```

The header is generated by cbindgen and is the same ABI the C#, Go, Java and R
bindings link against. It is the deterministic fact core only — no LLM, no network.
